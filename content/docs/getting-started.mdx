# Getting Started

Welcome to Reduxy.ai! This guide will help you get started with our privacy gateway for LLMs in just a few minutes.

## What is Reduxy.ai?

Reduxy.ai is a privacy gateway that sits between your application and AI providers like OpenAI, Anthropic, and others. It automatically detects and masks sensitive information (PII) in your requests, routes them intelligently, and provides comprehensive audit trails for compliance.

## Key Features

### Automatic PII Detection

Reduxy automatically detects and masks sensitive information including:

- **Email addresses**: `john@example.com` → `[EMAIL_1]`
- **Phone numbers**: `(555) 123-4567` → `[PHONE_1]`
- **Credit cards**: `4532 1234 5678 9012` → `[CREDIT_CARD_1]`
- **SSNs**: `123-45-6789` → `[SSN_1]`
- **IP addresses**: `192.168.1.100` → `[IP_1]`

### Request Headers

Reduxy adds special headers to track your requests:

- `X-Reduxy-Request-Id`: Unique identifier for each request
- `X-Reduxy-PII-Detected`: Number of PII entities detected
- `X-Reduxy-Processing-Time`: Time spent processing the request

## Example with PII

Here's what happens when you send a request containing PII:

```javascript
// Original request
const response = await fetch('https://api.reduxy.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4o',
    messages: [{
      role: 'user', 
      content: 'My email is john.doe@company.com and my phone is (555) 123-4567'
    }]
  })
});

// Reduxy sends this to OpenAI instead:
// "My email is [EMAIL_1] and my phone is [PHONE_1]"
```
